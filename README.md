# F360-Hackpad
So, as a user of CAD software such as Fusion 360 and SolidWorks, most of the time I only use a small range of shortcuts, so there is basically no need for a full-size keyboard, and a macro pad is enough for the job. So, I will use 16 MX switches and a 128Ã—32 OLED screen in order to determine the layer of the macropad. With that, I will not use a keyboard: I will use the first layer for shortcuts and the second layer for using numbers, so I will not need a keyboard.

# BOM :

1 SEED XIAO RP2040 (through-hole version)

1 0.91 inch 128*32 OLED Display

16 MX switches

16 DSA Keycaps

16 1N4148 Diodes

5 M3-16 screws

5 M3 heatset inserts
